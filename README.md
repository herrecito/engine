# Dependencies

* SDL 2.0
* SDL_image 2.0

# Build

If you want to take it for a spin, build with:

    make

Create a map with:

    ./bin/editor

(Left button to add walls, right button to delete, S to save, L to load)

And run it with:

    ./bin/engine

You'll need some textures and spritesheets:

* ascii.png
* ceil.png
* floor.png
* pistol.png
* wal.png

You can find mine here: http://imgur.com/a/F2Cnu
